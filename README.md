# SafeZone WiFi 🚀

**WiFi Threat Detection & Alert System (Lite Version)**

---

## 📖 Description
SafeZone WiFi is a lightweight Python tool to scan nearby WiFi networks and detect basic threats like:

- 🚨 Open (unsecure) WiFi Networks
- 🚨 Duplicate SSIDs (possible evil twin attack)
- 🚨 Very Weak Signal (possible rogue APs)

Simple alerts are shown on the terminal.
